## Compose Samples (Official)

These are the canonical examples from Google for learning Compose:

| Sample | Description |
|---|---|
| **JetNews** | News reader app — LazyColumn, top app bar patterns |
| **Jetsnack** | E-commerce — complex theming, animations |
| **Owl** | Onboarding flows, Compose animations |
| **Reply** | Adaptive layout — phone/tablet/foldable |
| **Now in Android** | Production-quality app with M3, adaptive, navigation |

All available at: https://github.com/android/compose-samples
